# 🛒 متجر عُجيل - Ogail Store

متجر إلكتروني متكامل ومميز مصمم لتقديم تجربة تسوق سلسة وسريعة للمستخدمين، مع لوحة تحكم شاملة لإدارة المتجر والطلبات والمنتجات. تم بناء المشروع بأحدث تقنيات الويب لضمان السرعة والأداء العالي والاستجابة الكاملة مع جميع الشاشات.

---

## 🚀 التقنيات المستخدمة (Tech Stack)

تم بناء المتجر باستخدام التقنيات التالية:

*   **الواجهة الأمامية (Frontend):**
    *   **[React 19](https://react.dev/):** لبناء واجهات مستخدم تفاعلية وقابلة لإعادة الاستخدام.
    *   **[Vite](https://vite.dev/):** كأداة بناء فائقة السرعة وخفيفة الوزن لتطوير وتجميع المشروع.
    *   **[Tailwind CSS](https://tailwindcss.com/):** لتصميم وتنسيق الصفحات بشكل عصري ومتجاوب (Responsive) بالكامل باستخدام نظام Grid و Flexbox.
    *   **[React Router DOM](https://reactrouter.com/):** لإدارة مسارات التنقل والروابط بين الصفحات بسلاسة ودون الحاجة لإعادة تحميل الصفحة.
    *   **[Lucide React](https://lucide.dev/):** لتوفير مجموعة واسعة من الأيقونات العصرية والجميلة.

*   **الخلفية وقاعدة البيانات (Backend & Database):**
    *   **[Firebase Firestore](https://firebase.google.com/docs/firestore):** قاعدة بيانات مرنة وسريعة في الوقت الفعلي (Real-time Database) لتخزين بيانات المنتجات، الطلبات، إحصائيات المبيعات، وإعدادات الشحن.
    *   **[Firebase Authentication](https://firebase.google.com/docs/auth):** لتأمين وحماية لوحة تحكم المسؤول (Admin Dashboard) وإدارة تسجيل الدخول.

## ✨ المميزات الرئيسية للمشروع (Key Features)

### 👤 1. واجهة العميل (Customer Storefront)
*   **الصفحة الرئيسية (Home Page):** تحتوي على لافتة ترويجية جذابة (Hero Section)، وعرض لأحدث المنتجات والأقسام، وقسم لآراء العملاء والضمانات.
*   **تصفح المنتجات (Products Catalogue):** استعراض جميع المنتجات مع إمكانية البحث والترتيب حسب السعر أو الاسم، وتصفيتها بناءً على الأقسام.
*   **تفاصيل المنتج (Product Details):** عرض معلومات تفصيلية عن كل منتج مع الصور المتاحة، الأحجام أو الخيارات المتوفرة، والتحكم بالكميات المطلوبة قبل الإضافة للسلة.
*   **سلة المشتريات التفاعلية (Dynamic Cart):**
    *   شريط جانبي سريع للسلة (Cart Drawer) لرؤية المنتجات المضافة فوراً.
    *   صفحة سلة مخصصة تتيح للعميل مراجعة المنتجات وتعديل الكميات أو حذفها مع حساب تلقائي للمجموع الإجمالي.
*   **إتمام الطلب (Checkout Page):** نموذج بسيط لجمع بيانات العميل (الاسم، رقم الهاتف، العنوان، المحافظة) مع حساب تكلفة الشحن تلقائياً بناءً على المحافظة المختارة.
*   **نجاح الطلب (Order Success):** تظهر للعميل فور إتمام الطلب وتزوده برقم تتبع فريد لمتابعة حالة الطلب.
*   **تتبع الطلب (Order Tracking):** يتيح للعميل إدخال رقم الطلب لمعرفة حالته الحالية (قيد الانتظار، تم الشحن، تم التوصيل، إلخ) مباشرة من قاعدة البيانات.

### 🔐 2. لوحة تحكم المسؤول (Admin Dashboard)
*   **تسجيل دخول آمن:** صفحة خاصة لتسجيل دخول مدير المتجر للوصول للوحة التحكم.
*   **الإحصائيات العامة (Dashboard Overview):** رسومات بيانية وتقارير ملخصة تعرض إجمالي المبيعات، عدد الطلبات، عدد المنتجات، والطلبات الأخيرة.
*   **إدارة المنتجات (Manage Products):** إمكانية إضافة منتج جديد (الاسم، السعر، الوصف، الصور، القسم، الحجم)، وتعديل أو حذف المنتجات الحالية في الوقت الفعلي.
*   **إدارة الطلبات (Manage Orders):** استعراض تفاصيل كافة طلبات العملاء وتغيير حالاتها (قيد المراجعة، جاري الشحن، تم التوصيل، ملغي) لتحديث العميل تلقائياً.
*   **إدارة تكاليف الشحن (Manage Shipping):** التحكم بأسعار الشحن وتخصيص تكلفة لكل محافظة لتسهيل عملية حساب الإجمالي تلقائياً للعميل أثناء الدفع.

## 🛠️ طريقة التثبيت والتشغيل المحلي (Setup & Run Locally)

لتشغيل هذا المشروع على جهازك المحلي، اتبع الخطوات التالية:

### المتطلبات الأساسية
*   يجب أن يكون لديك **Node.js** مثبتاً على جهازك (يفضل الإصدار LTS).

### الخطوات:

1.  **تحميل المشروع أو عمل Clone:**
    ```bash
    git clone https://github.com/your-username/Ogail_Store.git
    cd Ogail_Store
    ```

2.  **تثبيت الحزم والاعتمادات (Dependencies):**
    ```bash
    npm install
    ```

3.  **تثبيت وتشغيل بيئة التطوير المحلية:**
    ```bash
    npm run dev
    ```
    *سيظهر لك رابط محلي (غالباً `http://localhost:5173`) يمكنك فتحه في المتصفح لرؤية المشروع.*

4.  **بناء نسخة الإنتاج (Production Build):**
    عندما تريد رفع الموقع على خادم إنتاج، قم بتشغيل:
    ```bash
    npm run build
    ```
    *سيقوم الأمر بإنشاء مجلد باسم `dist` يحتوي على ملفات HTML/JS/CSS مضغوطة وجاهزة للنشر.*
